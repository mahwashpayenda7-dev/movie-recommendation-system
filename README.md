# 🎬 Movie Recommendation System

This is a simple movie recommendation system built using Python.

## 🚀 Features
- View all movies
- Rate movies
- Track watch history
- Get top-rated movies
- Get recommendations using BFS algorithm

## 🧠 Concepts Used
- Graph (Adjacency List)
- Breadth-First Search (BFS)
- Hash Tables (Dictionary)
- Queue (Deque)
- Sets

## 🔍 How It Works
Movies are connected based on shared genres.
We use a graph to represent these connections.
Then BFS is used to recommend similar movies.
